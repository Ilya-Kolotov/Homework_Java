# Задача. Реализовать функцию возведения числа а в степень b. a, b ∈ Z. Сводя количество выполняемых действий к минимуму.


## Декомпозиция:
1. Получение от пользователя число и степени
2. Запись полученных данных в файл input.txt
3. Вывести в консоль информацию о содержимом файла input.txt
4. Считать число и степнь из файла input.txt и произвести операцию возведения в степень
5. Запись полученного результата в файл output.txt
6. Вывести в консоль информацию о содержимом файла output.txt
